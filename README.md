## ⚒️ Usage

### Prerequisites

- [Node.js](https://nodejs.org) 20+
- [pnpm](https://pnpm.io) 9+

### Installation

1. Install dependencies: `pnpm install`
2. Generate Prisma: `pnpm run gen`
3. Run the website: `pnpm run dev`
